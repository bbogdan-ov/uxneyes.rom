# UXNEYES.rom

Like [XEyes] but for [UXN].

[XEyes]: https://cyber.dabamos.de/unix/x11/#xeyes
[UXN]: https://100r.co/site/uxn.html

![video](./video.gif)

## Play

To play UXNEYES.rom just download the [`output.rom`](./output.rom) file and open it with your
favorite UXN VARVARA emulator.

If you don't have one, you can use [the web emulator](https://rabbits.srht.site/uxn5/).

## Building

To build UXNEYES.rom first clone and compile [UXNSMAL] compiler.

[UXNSMAL]: https://github.com/bbogdan-ov/uxnsmal

```sh
uxnsmal eyes.smal
uxnemu output.rom
```
